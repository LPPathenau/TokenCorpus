# TokenCorpus

> Token术语语义演变语料库 | A Comprehensive Corpus of "Token" Semantic Variations

## 简介

TokenCorpus是计算机科学领域"Token"术语语义演变的专业语料库，支持计算词汇学、认知术语学研究。

**特性**：覆盖全 | 来源广 | 标注规范 | 开源共享

## 核心数据

| 指标 | 数值 |
|------|------|
| 记录数 | 30,000+ |
| 覆盖领域 | 7大领域 |
| 数据来源 | 20+类型 |

## 语义类别

- **T1**: 安全认证（API token, JWT）
- **T2**: NLP处理（tokenizer, word token）
- **T3**: 编译器（lexical token）
- **T4**: 区块链（crypto token）
- **T5**: AI/LLM（LLM token, context window）
- **T0**: 未分类/模糊

## 隐喻类型

- **M0**: 字面意义
- **M1**: 结构隐喻
- **M2**: 方位隐喻
- **M3**: 本体隐喻
