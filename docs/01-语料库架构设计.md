## 一、项目概述

### 1.1 项目目标

构建一个**覆盖全、来源广、标注规范、可开源共享**的计算机科学领域"Token"术语语义语料库，为计算词汇学、认知术语学、隐喻研究提供高质量的语言资源。

### 1.2 核心特性

| 特性 | 描述 | 目标 |
|------|------|------|
| **覆盖全** | Token在CS各领域的完整语义 | 7大语义类别全覆盖 |
| **来源广** | 国际标准、顶会论文、技术文档 | 10+数据源类型 |
| **标注规范** | 符合语料库语言学国际标准 | ISO 24622等标准 |
| **开源共享** | GitHub公开，支持学术引用 | CC BY 4.0许可 |

### 6.3 Inter-Annotator Agreement计算

```python
def calculate_icc(annotations: Dict[str, List[str]]) -> float:
    """
    计算标注者间信度 (Inter-Annotator Agreement)
    
    使用Krippendorff's Alpha作为主要指标
    支持：
    - 名义数据（语义类别）
    - 次序数据（隐喻类型）
    """
    from nltk.metrics.agreement import AnnotationTask
    import numpy as np
    
    # 转换数据格式
    data = []
    for item_id, annotators in annotations.items():
        for annotator, label in enumerate(annotators):
            data.append((annotator, item_id, label))
    
    # 计算Alpha
    task = AnnotationTask(data=data)
    alpha = task.alpha()
    
    return alpha
```
