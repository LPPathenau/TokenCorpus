## 标注一致性检验

请标注以下句子，与黄金标准对比：

### 练习1
```
"The refresh token has a longer expiration time than the access token."
```
<details>
<summary>点击查看答案</summary>

**答案**: T1d (refresh token) + M0  
**理由**: refresh token用于刷新access token，属于安全认证类
</details>

### 练习2
```
"The vocabulary contains 50,000 unique tokens."
```
<details>
<summary>点击查看答案</summary>

**答案**: T2a (NLP词汇单元) + M0  
**理由**: 讨论tokenizer的词汇表大小
</details>

### 练习3
```
"Each token represents a semantic unit of meaning."
```
<details>
<summary>点击查看答案</summary>

**答案**: T2a (语义单元) + M1 (本体隐喻)  
**理由**: 将token概念化为"语义单元"，属于本体隐喻
</details>
